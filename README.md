# Titanic Data Exploration 🚢

This project analyzes the Titanic dataset using **Pandas** and **NumPy** to extract meaningful insights.

## 📌 Project Overview
The goal of this project is to perform data exploration and understand patterns such as:
- Survival rate
- Gender-based survival comparison
- Passenger class distribution
- Age analysis
- Correlation between different features

## 📂 Dataset
The dataset used is the Titanic dataset containing passenger details like:
- PassengerId
- Name
- Age
- Sex
- Pclass
- Fare
- Survived

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Jupyter Notebook

## 📊 Key Insights
- Females had higher survival rates than males.
- Passengers in 1st class had better survival chances.
- Children had relatively higher survival rates.

## ▶️ How to Run
1. Clone the repository
2. Open the Jupyter Notebook:
   ```
   Titenic_Dataset_Insight.ipynb
   ```
3. Run all cells to see analysis.
