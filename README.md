# Titanic Dataset Insights Using Pandas & NumPy 🚢

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python libraries like Pandas and NumPy.

---

## 📌 Objective

To analyze passenger data and discover patterns that influenced survival during the Titanic disaster.

---

## 📂 Dataset Information

The dataset includes details such as:

- PassengerId
- Pclass (Ticket Class)
- Name
- Sex
- Age
- SibSp (Siblings/Spouses aboard)
- Parch (Parents/Children aboard)
- Fare
- Embarked
- Survived (Target Variable)

---

## 🛠 Tools & Libraries Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## 📊 Analysis Performed

✔ Data Cleaning  
✔ Handling Missing Values  
✔ Survival Rate Analysis  
✔ Gender-wise Survival Comparison  
✔ Class-wise Survival Comparison  
✔ Age Distribution Analysis  
✔ Statistical Summary  

---

## 📈 Key Insights

- Female passengers had a significantly higher survival rate.
- 1st class passengers had better survival chances.
- Children had relatively higher survival probability.
- Fare and survival show some positive correlation.
